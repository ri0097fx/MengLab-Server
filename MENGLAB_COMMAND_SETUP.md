# `menglab` コマンド化ガイド

## macOS / Linux

```bash
chmod +x ./install_menglab_unix.sh
./install_menglab_unix.sh
```

インストーラ実行時に PATH 追加を確認します（デフォルト No）。

```bash
menglab
```

```bash
menglab setup
menglab preview about/
menglab reset
```

---

## Windows (PowerShell / CMD)

```powershell
powershell -ExecutionPolicy Bypass -File ".\install_menglab_windows.ps1"
```

インストーラ実行時に PATH 追加を確認します（デフォルト No）。

```powershell
menglab
```

```powershell
menglab setup
menglab preview about/
menglab reset
```
